# Tesla Website Replication

This project replicates the Tesla website using React (Vite) for the frontend.
  
## Tech Stack
- React + Vite
- Tailwind CSS (for styling)
- Node.js + Express (for backend authentication)
- MongoDB (for user + session data)

## Setup Instructions
1. Clone this repo
2. Run `npm install`
3. Run `npm run dev` to start the project
